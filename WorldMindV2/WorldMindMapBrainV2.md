# WorldMindMapBrainV2 Hosted Icon List

Paste hosted PNG/WebP URLs into `Ui.HostedButtonImages` in `oxide/config/WorldMindMapBrainV2.json`.

If a URL is blank, the UI falls back to clean 3-letter text. No empty icon squares. No tiny overlay letters on images.

Recommended icons should be square, transparent PNG/WebP, 128x128 or 256x256.
